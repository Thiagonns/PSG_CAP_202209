# Projeto Avaliar

Projeto realizado como forma de avaliação final da capacitação PSG (2022-09).

## Estrutura

Abaixo estão relacionadas as tecnologias utilizadas.

* SQL Server
* C#
* .NET
* JavaScript
* JQuery
* HTML
* Bootstrap

## Versão

Foi utilizada a versão 6.0 do .NET e a versão 4.0 do Bootstrap.
